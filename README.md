# Ninjo

A cozy ninja platformer full of leafy forests, sneaky enemies, and speedy parkour mayhem.

## Game Highlights

- **Fast-paced ninja movement** with wall jumps, slides, and dashing through levels.
- **Vibrant pixel-style terrain** built from tilemaps and charming animated sprites.
- **Dynamic obstacles and enemies** that keep every run exciting.
- **Fun particle effects** for leaves, sparks, and motion.

## How to Run

1. Install the requirements:
   ```bash
   pip install -r requirement.txt
   ```
2. Launch the game:
   ```bash
   python main.py
   ```

## Controls

- **Arrow keys / WASD** — Move and jump
- **Space** — Jump
- **Down** — Slide or drop down
- **Left / Right** — Run and navigate

## Project Structure

- `main.py` — Launches the game.
- `main_menu.py` — Main menu and title screen.
- `game.py` — Game loop, state handling, and world updates.
- `editor.py` — Level editor / debug helper.
- `scripts/` — Contains tilemap, entity, particle, and utility logic.
- `data/` — Game maps, sprite assets, and sound effects.
